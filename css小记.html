input 的 disabled 和 readonly 区别？

disabled:禁用，置灰，提交input表单时候不会被提交,可以作用于任何 input 类型的表单（select radio text checkbox 。。） readonly：只读，不置灰，提交表单数据仍然可以被提交，只作用于 type 为input(text / password)和textarea
